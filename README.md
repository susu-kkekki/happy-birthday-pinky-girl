# Vulani's Birthday Project

A fun interactive birthday web page designed as a themed "case file" mystery. The experience guides the user through three evidence cards, and once all answers are verified, a special birthday message is revealed.

## What this project does

- Presents a stylized detective-style interface
- Includes three verification cards with puzzle-style questions
- Reveals a hidden birthday message after all sections are completed
- Uses HTML, CSS, and JavaScript for the full experience

## Project files

- index.html — main structure of the page
- style.css — visual styling and responsive layout
- script.js — answer verification logic and final reveal
- vulani.jpeg — image shown in the final section

## How to run

1. Open index.html in your browser, or
2. Serve the folder with a simple local server if you prefer.

No build step is required.

## Notes

This project is a playful birthday surprise and can be customized by editing the questions and answers in script.js, and editting the image and birthday message in index.html.
